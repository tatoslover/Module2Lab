Lab Exercise 1: HTML Fundamentals

Goal: To practise using common HTML tags and become familiar with best practice usage for building a modern web page.

  • Practice HTML metadata (title, meta tags for charset, description, keywords, author, viewport)
  • Work with HTML elements: div, span, p, pre, ul, li, ol, article
  • Create tables with thead, tbody, tr, td, th, colspan, rowspan
  • Add multimedia: img, video, audio with proper attributes
  • Build forms with input, label, select, option, button elements
  • Add navigation links between pages
  • Use browser inspection tools to examine HTML structure

Extension: Set up a mock testing back-end server and link it to your form to receive submitted data.

Lab Exercise 2: CSS Basics

Goal: To practise using common CSS selectors and rules and become familiar with best practice usage for styling a responsive web page.

  • Selectors & Combinators: Type, class, ID, descendant, and child combinators
  • Box Model: Margin, padding, border, width, height, box-sizing, overflow
  • Positioning: Absolute, relative, and sticky positioning
  • Typography: Colour, font-size, line-height, font-weight, text alignment
  • Link States: Styling :hover, :active, :focus, :visited states
  • Modern CSS: calc() function and dynamic calculations

Extension: Test your knowledge with the CSS Selector Game and practice styling a biography page.

üìê Lab 3: Flexbox Layout System

‚ñº
Lab Exercise 3: Flexbox

Goal: To become familiar with using flexbox as a layout model to style responsive web pages.

  • Basic Flexbox: Set flex-container display and flex properties on items
  • Centring: Use justify-content and align-items to centre elements
  • Space Distribution: Apply space-between and other justify-content values
  • Wrapping Items: Use flex-wrap and flex-basis for responsive layouts
  • Direction Control: Experiment with flex-direction values
  • Navigation Menus: Create horizontal navigation with flex properties

Extension: Choose a project from Frontend Mentor to implement using flexbox techniques.

Key Learning: Flexbox revolutionizes layout design by providing intuitive, flexible solutions for alignment, distribution, and responsive behaviour without complex calculations.

Lab Exercise 4 & 5: Build Your Own Responsive Website

Lab 4 Goal: To practise using HTML and CSS to build a real-world simple one-page resume website.

Lab 5 Goal: To practise using responsive HTML and CSS to build a real-world simple one-page resume website.

  • Start from scratch - don't use a template
  • Sketch a design or find a style to copy
  • Build HTML structure and style with CSS in separate files
  • Use all learned skills: Semantic HTML, sections, flexbox, images, box model, typography
  • Add media queries to adjust for smaller devices like phones
  • Ensure no horizontal scrolling on mobile devices
  • Consider responsive adjustments: font-size, flexbox direction, alignment

Extension: Add advanced features by copying elements from other professional resume websites or try a Frontend Mentor project.
